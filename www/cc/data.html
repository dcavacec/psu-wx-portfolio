This page contains drop down menus which allows various data scatterplots to be viewed. The data is obtained from the Paul Kossin dataset and includes intensity vs. time and various wind radii. The scatterplots can be viewed on a yearly basis; each storm in a year has its individual scatterplot.

     This page also contains all of the hurricane tracks and storm intensities for every year in the Atlantic Basin. All storms are overlayed on one map for each year. A legend appears on each map depicting the corresponding color for each storm intensity category.
