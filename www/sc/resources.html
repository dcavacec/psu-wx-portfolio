IT in Meteorology Resources

Perl
ActivePerl

My choice Perl distribution is ActiveState's `ActivePerl. Download the source code and manual from their site, and check out their additional links too.

CPAN

The Comprehensive Perl Archive Network. One of the major advantages of Perl is its strong community. This archive network contains a multitude of modules to aid you in your various tasks. For example, the "WWW::Mechanize" module makes browsing the web with Perl quite simple. The Date::Simple module can help you in creating a list of dates in a defined format, which is deceptively tricky. Utilizing this archive is key because its exposes you to many different facets of the language, it forces you to familiarize yourself with object-oriented programming, and it is the core of the power behind the Perl language.

Meteo 473

Dr. Clothiaux's Meteo 473 homepage. Contains Perl bootcamps, lectures from Dr. Harrington and Dr. Young, milestone descriptions and examples, course outlines and additional resources. This is a great place to get a feel for the expectations of the class before you enroll.

O'Reilly Media

O'Reilly Books are the Perl texts of choice. Check out "Programming Perl," "Learning Perl," and "Perl Cookbook." FULL TEXT AVAILABLE THROUGH SAFARI COMPUTER BOOKS ONLINE!

perl.org
perl.com

Competing websites for documentation resources. Each is filled with a plethora of excellent learning tools.

perlmonks

A Perl community site. Includes a vast tutorial and FAQ database, an instructional forum, a chat dialogue, a scratchpad for saving code snippets, and tons more. Fully customaizable page layout after signing up. I spent days chatting with coders and posting my questions while learning about the language. HIGHLY RECOMMENDED!

Safari Computer Books Online

Don't skip this one. Most of the O'Reilly Perl books are available free online, cover to cover. This is an enormous value as these books usually go for over $40 new.

WWW::Mechanize Tutorial (Browser Agent)

A walkthrough about the simplest way to browse, scrape, or spider the web with Perl. It discusses installation of modules from CPAN, object-oriented programming (LEARN ABOUT IT), the Data::Dumper module (great for debugging), and more.

Fortran
g95

g95 is a Fortran 95 compiler compatable in most of the major operating systems. From the Downloads page, download and install the source code pertaining to your OS. Then get the manual, the "Getting Started" file, and the "Running g95" file from the Documents page. Other useful links can be found on the site as well, such as a FAQs page and training resources.

www.tek-tips.com

Another great forum full of professionals concerned with a plethora of languages and applications. I found that my questions posted in the Fortran forum were typically answered in 1-3 days. Make sure to follow all of the guidelines when posting questions, as it is stated clearly upon signing up that the site is intended for professionals (Don't get pinned as a student trying to suck answers to homework problems. Always provide background info, details about the problem, what you've tried, your code and the error(s) you're experiencing).

Introductory Fortran Lecture

This is a detailed presentation from the University of Cambridge created for Fortran newcomers. It starts with a general intro to the language and is followed by explanations of the inherent rules, data types, flow controls, array constructs, etc.

Fortran 95 Text

A pdf document that goes into great detail about the language. Examples are included after each chapter. I personally feel that this document was much better at getting me comfortable with Fortran than the g95 manual.

VBA Excel
excel-vba.com

This step-by-step walkthrough will get you up and running with VBA in no time. It starts out with an explanation of the VBA Editor, which is where you will edit the VBA code for Excel. By then building and testing macros through the VBAE and comparing the procedure with recording macros in Excel, the user quickly realizes how easy automation in Excel can be with the combined use of each interface.

ozgrid.com

Tons of downloadable VBA code, macros, and examples. Hit the tutorial above first!

MySQL
Building a MySQL Database

An introduction to building a database with MySQL. Basic, stright forward, and great for learning quickly.

Weblion MySQL Tutorial

A PSU resource for MySQL beginners.

Sun Microsystems

MySQL is open-source (free) and can be downloaded directly from the developer's website. Community Server and other tools are available.
