Scatter plot Data

We have 5 different types of charts and scatter plots that give details of every tropical storm and hurricane from 1983-2005. We wrote a matlab program using Jim Kossin’s hurricane data set to achieve this task. The 5 plots are: intensity Vs time, max wind radius Vs intensity, gale force radius Vs intensity, 50 knot wind radius Vs intensity, and 64 knot radius Vs intensity. These plots can be accessed by clicking the data link in the navigation bar above or by clicking here. Each plot contains every storm for one particular year. We also fit a regression line to the max wind radius and gale force radius charts. These regression lines can be used to see any correlation between these wind radius's and the intensity of the storm.

Hurricane Forecast Models

We also used Matlab and Weka to develop five statistical models that predict 3 hours into the future. We are able to predict: 3 hr intensity change, 3 hr latitude change, 3 hr longitude change, 3 hr max radius change, and sea surface temperature. Click on the model of choice for a more in depth analysis and description. Forward and backward differencing was utilized to uncover the changes between these variables over a three hour time interval.  This massive swath of data was organized into comma separated value (CSV) files and displayed on the Penn State meteorological server for convenient student access.  To further analyze this data, the variables were input into WEKA, which is a statistical program.  A detailed table was created to summarize outcomes from various data statistic models and decision trees in WEKA. Both hurricane statistics and synoptic weather variables proved to be useful predictors for forecasting hurricane track and intensity, the predictands.
